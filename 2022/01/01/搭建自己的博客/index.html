搭建自己的博客

搭建博客需要的东西

  • git
  • node.js
  • nvm：node 包管理工具（非必须）
  • GitHub 账号
  • mpm

1.搭建前准备：

  1. 安装
  • git 下载地址：https://git-scm.com/
  • nodejs(稳定版本，不要下载最新版) 下载地址：https://nodejs.org/en

2.开始搭建

  1. 在电脑目录下创建一个 blog 目录
    mkdir blog

  2. 安装 hexo-cli
    npm install -g hexo-cli

  3. cd blog目录下初始化 hexo

  4. 查看是否成功
    hexo s
    ![source_posts/images01/检查hexo初始化是否成功.png]

3.将 Hexo 部署到 Github

  1. Github 创建个人仓库

  2. ![source_posts/images01/创建仓库.png]

  3. 生成 ssh 添加到 Github

    • 将生成的公钥文件里面的公钥添加 Github -> setting ->SSH ang GPG keys 中

  4. 修改配置文件_config.yml

  5. 安装 deploy-git
    npm install hexo-deployer-git --save

  6. 依次执行以下命令
    hexo clean: 清除缓存文件 db.json 和已经生成的静态文件 public
    hexo g: hexo generate的缩写生成，网站静态文件默然设置的 public 文件夹
    hexo d: hexo deploy的缩写，自动生成网站静态文件，并部署到设定是仓库

    • 注意hexo ddeploy 时需要输入 yes
    • 查看仓库是否上传成功
    • 可以使用自己配置的仓库名字直接进入https://auroramu.github.io/
    • hexo new "搭建自己的博客"
