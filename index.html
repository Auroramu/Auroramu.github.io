JavaScript数组API

Array(API)

  1. Array.length:返回数组的长度
  • 修改原数组的方法
  1. Array.push():将一个元素和多个元素添加到数组的末尾；默认返回数组的长度,改变原来的数组，可以添加数组类型
  1. Array.pop():默认删除并返回数组的最后一个元素,没有参数，改变原数组
  1. Array.unshift():将一个或者多个元素添加到数组的开头，并返回数组的长度
  1. Array.shift():默认删除并返回数组开头的第一个元素，没有参数，改变原数组
  1. Array.splice():从指定位置删除，替换或者添加元素，并返回被删除的元素

返回新数组的方法

  1. Array.concat:将两个或者多个数组合并成一个数组，并返回结果，不改变原数组
  1. slice():返回指定索引范围的元素，不会修改原数组,返回新数组
  1. filter():创建一个新数组,其中包含满足条件的元素
  1. map(): 创建一个新数组，其中的元素为原始数组经过指定函数的处理的结果
  1. reverse():返回的是新数组，反转数组中元素的顺序
  1. sort():对数组的元素进行排序

元素定位方法

  1. indexOf():返回元素在数组中首次出现的索引，如果不存在则返回-1
  1. lastIndexOf():返回元素在数组中最后一次出现的索引，如果不存在返回-1
  2. arr.lastIndexOf(34) //返回索引位置

arr.includes(34) //返回布尔值

arr.join(“-“)

arr.toString()

arr.reduce((accumulator,currentValue)=>accumulator+currentValue)

arr.reduceRight((accumulator,currentValue)=>accumulator+currentValue)

搭建自己的博客

搭建博客需要的东西

  • git
  • node.js
  • nvm：node 包管理工具（非必须）
  • GitHub 账号
  • mpm

1.搭建前准备：

  1. 安装
  • git 下载地址：https://git-scm.com/
  • nodejs(稳定版本，不要下载最新版) 下载地址：https://nodejs.org/en

2.开始搭建

  1. 在电脑目录下创建一个 blog 目录
    mkdir blog

  2. 安装 hexo-cli
    npm install -g hexo-cli

  3. cd blog目录下初始化 hexo

  4. 查看是否成功
    hexo s
    ![source_posts/images01/检查hexo初始化是否成功.png]

3.将 Hexo 部署到 Github

  1. Github 创建个人仓库

  2. ![source_posts/images01/创建仓库.png]

  3. 生成 ssh 添加到 Github

    • 将生成的公钥文件里面的公钥添加 Github -> setting ->SSH ang GPG keys 中

  4. 修改配置文件_config.yml

  5. 安装 deploy-git
    npm install hexo-deployer-git --save

  6. 依次执行以下命令
    hexo clean: 清除缓存文件 db.json 和已经生成的静态文件 public
    hexo g: hexo generate的缩写生成，网站静态文件默然设置的 public 文件夹
    hexo d: hexo deploy的缩写，自动生成网站静态文件，并部署到设定是仓库

    • 注意hexo ddeploy 时需要输入 yes
    • 查看仓库是否上传成功
    • 可以使用自己配置的仓库名字直接进入https://auroramu.github.io/
    • hexo new "搭建自己的博客"
